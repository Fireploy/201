    Function showSocialNetworks

    • showSocialNetworks(redSocial: RedSocialUsuario, props?: SvgIconProps): Element[]

      Renders a list of social network icons based on the user's social media data.

      This function iterates over the redSocial object, checking each social media key for valid (non-null and non-empty) URLs. For each valid social network, it renders the corresponding social media icon.

      Parameters

      • redSocial: RedSocialUsuario

        An object containing the user's social media profiles, where keys are social media platform names (e.g., "facebook", "instagram") and values are URLs to the user's profiles.

      • Optionalprops: SvgIconProps

      Returns Element[]

      An array of JSX elements representing each social network icon, which are displayed based on the available social media profiles.

      Example

      const redSocial = {
        facebook: "https://facebook.com/juanperez",
        instagram: "https://instagram.com/juanperez",
        linkedin: "",
        x: "https://x.com/juanperez"
      };
      const socialIcons = showSocialNetworks(redSocial);
      // This will return JSX with SocialNetworkIcon components for Facebook, Instagram, and X.
